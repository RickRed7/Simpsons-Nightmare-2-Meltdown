# Simpsons-Nightmare-2-Meltdown
Bart wakes into a 16-bit fever dream where Springfield is melting into digital chaos. This "Meltdown Edition" pits you against Nelson’s glitch-horde and reality-warping static. If released in '94, its "Overwriting" mechanic would’ve redefined SNES limits, turning the console’s own memory against you in a psychedelic glitch-war.
